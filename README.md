# 🎓 Student Management System

A simple Student Management System built using PHP and MySQL with a clean Bootstrap UI.  
It helps manage student records with basic CRUD operations.

---

## 🚀 Features

- ➕ Add Student
- 📋 View Students List
- ✏️ Edit Student Details
- ❌ Delete Student
- 🔍 Search Students (Name / Email / Course)
- 📊 Total Students Dashboard Counter
- 🎨 Responsive UI (Bootstrap)

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, Bootstrap
- Backend: PHP
- Database: MySQL
- Server: XAMPP (Apache)

## ⚙️ Setup Instructions

### 1. Install XAMPP
Download XAMPP:  
https://www.apachefriends.org/

---

### 2. Move Project Folder
Place project inside:


C:\xampp\htdocs\student-management-system

3. Start Server

Open XAMPP Control Panel:

Start Apache
Start MySQL

4. Create Database

Open:


http://localhost/phpmyadmin


Create database:


student\_db


Import your SQL table (if available).

5. Run Project

Open browser:


http://localhost/student-management-system/
